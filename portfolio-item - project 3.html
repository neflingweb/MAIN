Fourth Year Individual Project Space Attack

Project Description

I decided to go with an Android game because I have a high interest in video games and their development. Developing a game will be a step up from my previous projects. I choose Android because my 3rd year project was developed on that platform and the skills I have aquired from my previous projects will be useful in this area. The game engine that I picked is Unity and the main reason for that is that I tried using it before and it was very intuitive and easy to work with. The project idea is to create an android based game with a cloud database to store user information. The theme is a space based game, in which the player takes control over a ship in order to defeat enemies that will get stronger as the player progresses through the levels of the game. The user will be able to upgrade his spaceship to make it easier to reach higher levels. To increase the complexity of the program I will add an in app purchase option to remove ads from the application.

Project Details
