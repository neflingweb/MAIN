Team Project Third Year X-League

Project Description

IT Tallght second year computing project. Team project application Milkrun. The purpose of this program was to help improve a milkman's job. The milkmen usually use logbooks and physical notes to what and when to deliver the goods. This program was made to speed up the process of looking for orders and adding new orders to the list. Project done in Java with an SQL database.

Project Details

  • Java based
  • SQL Database
  • Team Size: 3
